# Insurance-Cost-Analysis

Analyze and predict insurance costs using real-world data! This project walks you through cleaning, exploring, and modeling an insurance dataset to understand what drives annual charges.

Key Highlights:

Perform data cleaning and handling of missing values

Conduct exploratory data analysis (EDA) to uncover trends and insights

Build single-variable and multi-variable Linear Regression models

Improve predictions with Ridge Regression

Dataset includes: age, gender, BMI, number of children, smoker status, region, and annual charges.

A concise, hands-on project for data analysis and predictive modeling practice.
